# Estudos de Lógica em Java ☕

Este repositório contém exercícios práticos de lógica de programação utilizando Java.

##  Projeto: Calculadora de Proporção
Um script simples que automatiza o cálculo de valores totais (100%) a partir de uma base percentual informada. 

**Exemplo de uso:**
- Entrada: 1510.53 (referente a 30%)
- Resultado: 5035.10 (referente a 100%)

##  Tecnologias
- Java 21
